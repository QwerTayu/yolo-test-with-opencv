## virtualenvコマンド
### プロジェクトの作成
python3 -m virtualenv 仮想環境名
### 仮想環境の起動
source 仮想環境名/Scripts/activate
### シャットダウン
deactivate 
### パッケージのインストール
python -m pip install requests
